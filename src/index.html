The BackBit Tool assembles .BBT files out of existing program, disk and binary resources.
LEVEL I: Startup Program
Not required, but if not present then a mounted disk must be present.
*.PRG
LEVEL II: Mounted Disks
Can mount up to 8 disks. If no startup program is present, the program will be started with LOAD "*",8,1.
*.D64 *.D71 *.D81
LEVEL III: Extended Data
Optional. Add raw binary data, accessible in 64K chunks. Total BBT file size must be under 4GB.
